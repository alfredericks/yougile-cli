# Yougile CLI

Интерактивный CLI клиент для управления задачами в [Yougile](https://yougile.com).

## Установка

```bash
npm install -g yougile-cli
```

Или локально:

```bash
npm install
npm run build
npm link
```

## Настройка

Перед началом работы настройте CLI:

```bash
yougile init
```

Есть два способа авторизации:

1. **Email + пароль** (рекомендуется) — CLI автоматически создаст API ключ
2. **Существующий API ключ** — если он у вас уже есть

При авторизации через email/пароль:
- Введите email и пароль от Yougile
- Выберите компанию (если их несколько)
- API ключ создастся автоматически

При настройке можно выбрать проект, доску и колонку по умолчанию — тогда создание задач будет быстрее.

## Использование

### Создание задачи (интерактивный режим)

```bash
yougile create
```

или короткая версия:

```bash
yougile c
```

CLI спросит:
1. Куда положить карточку? (проект → доска → колонка)
2. Название задачи
3. Описание (опционально)
4. Дедлайн (опционально)
5. Ответственный (опционально)

### Быстрое создание (с дефолтами)

Если настроены значения по умолчанию:

```bash
yougile create --quick
# или
yougile c -q
```

Задаст только название задачи, остальное возьмёт из дефолтов.

### Создание с параметрами

```bash
yougile create --title "Название задачи" --description "Описание"
```

### Просмотр задач

```bash
yougile list      # задачи из дефолтной колонки
yougile ls -a     # все задачи
```

### Конфигурация

```bash
yougile config    # просмотр и редактирование настроек
```

## Конфигурационный файл

Настройки хранятся в `~/.config/yougile/config.json`:

```json
{
  "apiKey": "your-api-key",
  "apiHost": "https://yougile.com/api-v2/",
  "defaultProjectId": "...",
  "defaultProjectName": "My Project",
  "defaultBoardId": "...",
  "defaultBoardName": "Main Board",
  "defaultColumnId": "...",
  "defaultColumnName": "To Do"
}
```

## Команды

| Команда | Алиас | Описание |
|---------|-------|----------|
| `yougile init` | | Начальная настройка |
| `yougile create` | `c` | Создать задачу |
| `yougile list` | `ls` | Список задач |
| `yougile config` | `cfg` | Настройки |

## Опции create

| Опция | Описание |
|-------|----------|
| `-t, --title <title>` | Название задачи |
| `-d, --description <desc>` | Описание |
| `-q, --quick` | Быстрый режим с дефолтами |

## Требования

- Node.js >= 18.0.0
- API ключ Yougile
